1、calendar

日历小程序。

用于打印某个月的日历，输入 1900 年以后的时间即可。



2、char_cnt

字符统计。

统计给定字符串中的各类字符的个数。



3、img_rcg

读取一张黑白位图，并给出图中白色区域的重心。

linux 下需要把 DWORD 的定义改成 unsigned int，在 windows 下用 unsigned long 是可以的。



4、score_cnt

成绩统计。

统计给定分段内的分数个数。



5、infraredwork

红外技术大作业。

黑体辐射照度计算。
